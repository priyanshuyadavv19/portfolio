# Priyanshu Kumar Yadav - Portfolio
A personal portfolio website built with plain HTML, Tailwind CSS and vanilla JavaScript.

## Features
* Smooth Scrolling
* Active navbar link highlighting
* Hamburger menu
* Fully Responsive Design

## Tech Stack Used
* HTML
* Tailwind CSS
* JavaScript
